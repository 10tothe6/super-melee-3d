For anyone who has played the 1992 sci-fi arcade game Star Control II, you may recognize this project as the Super Melee gamemode re-imagined in three dimensions. This was developed over the course of a two-week period.

## Installation:
Simply download the .zip files from the latest version and extract them.

## Starting A Server:
Run the .exe file for the server, then you can run the .exe file for the client and connect to it using your IP address. There is no limit on the number of players per game, connect as many clients as you want.
